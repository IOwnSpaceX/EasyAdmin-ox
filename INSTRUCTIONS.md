Instructions for simple EasyAdmin-ox install.
By: Liam & Saxon

1. Please install the file from the GitHub - https://github.com/IOwnSpaceX/EasyAdmin-ox/tree/main
2. Click the GREEN button labled as "Code"
3. Click "Download Zip" from the dropdown
4. Drag the newly installed "EasyAdmin-ox-main" file to your server resources folder
5. RENAME the file from "EasyAdmin-ox-main" to "EasyAdmin-ox" [IMPORTANT]
6. Locate your "server.cfg" file and open it
7. In the file type "ensure EasyAdmin-ox"
8. Add your server permissions using ACE Perms - You can find all ACE Perms here: https://easyadmin.readthedocs.io/en/latest/permissions/ and the modded perms for this menu in the README.md
9. Restart your server and if everything is successful the resource will start in your server console!

FOR OPTIONAL DUTY SYSTEM
Please add "set ea_DutySystemEnabled true" to your "server.cfg" to enable the duty system!

EXAMPLE IN SERVER.CFG (DO NOT add quotation marks!): set ea_DutySystemEnabled true
